# Matrix CLI Application

A C# command-line application for performing a wide range of matrix operations, featuring a clean TUI powered by **Spectre.Console**.  
The program supports matrix creation, storage, manipulation, and advanced linear algebra operations such as determinants, LU decomposition, and matrix inversion.
